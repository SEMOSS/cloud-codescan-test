# Hello World – React + Java Servlet + Tomcat

A "Hello World" app with a React frontend and a Java Servlet backend, served by Apache Tomcat inside a Docker container. The message is generated by the Java backend and fetched by React at runtime.

## Stack

| Layer     | Technology                        |
|-----------|-----------------------------------|
| Frontend  | React 18 (pnpm)                   |
| Backend   | Java 17 Servlet (`/api/hello`)    |
| Server    | Apache Tomcat 10.1 (JDK 17)       |
| Build     | Maven 3.9                         |
| Container | Docker (3-stage build)            |

## Build & Run

```bash
# 1. Build the Docker image
docker build -t hello-world-react .

# 2. Run the container
docker run -p 8888:8888 hello-world-react
```

Open [http://localhost:8888](http://localhost:8888) in your browser.

## How it works

1. **Stage 1 (Node 20 / pnpm)** – installs dependencies and runs `react-scripts build`, producing an optimized static bundle.
2. **Stage 2 (Maven 3.9 / JDK 17)** – compiles `HelloServlet.java` and packages everything (servlet + React static files) into a single `hello-world.war`.
3. **Stage 3 (Tomcat 10.1 / JDK 17)** – deploys the WAR as `ROOT`. The custom `server.xml` sets the HTTP connector to port **8888**.

## Request flow

```
Browser → GET /          → Tomcat serves the React SPA
React   → GET /api/hello → HelloServlet returns {"message":"Hello World"}
React renders the message on screen
```
